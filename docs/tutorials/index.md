## Overview

The OPS Explorer is a multi-panel workspace where you can:

- **Search gene knockouts** by name and instantly see phenotypic fingerprints
- **View synchronized visualizations** — embeddings, volcano plots, and fluorescence image galleries side by side
- **Explore multimodal data** — OPS morphology and CROP-seq transcriptomics linked at the perturbation level
- **Download subsets** of data for follow-up analysis in your own tools

Each tile above links to guides tailored to different needs: quick orientation or comprehensive reference.
